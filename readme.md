# AITests

Projet tres simple pour centraliser mes essais autour de l'IA et du web.

## Objectif

- Avoir un endroit unique pour faire des tests IA (prompts, agents, skills, instructions).
- Construire et maintenir un petit site web pouvant etre heberge facilement.

## Stack visee

- Frontend: Astro (statique)
- Backend: PHP simple
- Base de donnees: MySQL ou MariaDB
- Deploiement: hebergement partage

## Utilisation

Ce repo sert de bac a sable pratique:

- tester des idees IA rapidement,
- organiser les fichiers d'instructions/agents,
- preparer un site web simple et hebergeable.
